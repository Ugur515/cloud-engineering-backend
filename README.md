# Cloud Engineering Backend Service

Dieses Projekt ist ein cloud-fähiger Backend-Service, der grundlegende Konzepte
des Cloud Engineerings demonstriert, wie saubere Architektur,
Konfigurationsmanagement und Infrastruktur als Code.

Der Service wird lokal entwickelt und kann auf Cloud-Anbietern wie AWS oder Azure
deployt werden.

## Projektstatus
In Arbeit – grundlegende Struktur wird aufgebaut.

## Technologien
- Python
- Flask
- REST
- Git

## Ziele
- Sauberen, cloud-fähigen Backend-Service entwickeln
- Praktische Kenntnisse im Cloud Engineering erlangen
- Vorbereitung auf Junior Cloud / Backend Rollen
